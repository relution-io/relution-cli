### Dev Guide
---

##### Installation
```bash
$: git clone git@github.com:relution-io/relution-cli.git
$: cd relution-cli
$: npm i -g tsconfig-cli typings istanbul
$: npm i
$: npm run watch
```

##### Test

```bash
$: npm test
```

##### Api reference
```bash
$: npm run api
```
